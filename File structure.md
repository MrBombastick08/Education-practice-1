# Структура файлов проекта

Основной framework - Flask

| Файл/Директория | Назначение |
| :--- | :--- |
| **`app.py`** | **Главный файл приложения.** Содержит инициализацию Flask, конфигурацию, определение основных маршрутов (роутов) и логику обработки HTTP-запросов. |
| **`config.py`** | **Конфигурация приложения.** Хранит настройки Flask, такие как `SECRET_KEY` и `SQLALCHEMY_DATABASE_URI` для подключения к базе данных. |
| **`models.py`** | **Модели данных (ORM).** Определяет классы Python, которые соответствуют таблицам в базе данных (Product, Workshop, MaterialType и т.д.) с использованием Flask-SQLAlchemy. |
| **`forms.py`** | **Веб-формы.** Содержит классы форм (например, `ProductForm`, `MaterialCalculatorForm`), используемые для сбора и валидации данных от пользователя. |
| **`services.py`** | **Бизнес-логика.** Вынесенные функции для выполнения сложных расчетов, таких как `calculate_production_time` (расчет времени производства) и `calculate_raw_material` (расчет сырья). |
| **`init_db.py`** | **Скрипт инициализации.** Используется для создания всех таблиц в базе данных при первом запуске и загрузки начальных данных из Excel-файлов. |
| **`DataBase.sql`** | **Схема БД (SQL).** Содержит SQL-скрипт для создания таблиц, может использоваться для ручной инициализации или миграций. |
| **`sql_table_picture.pdf`** | **Визуальная схема БД.** Графическое представление структуры базы данных. |
| **`templates/`** | **Шаблоны HTML.** Директория для хранения Jinja2 шаблонов, которые используются для генерации пользовательского интерфейса (например, `index.html`, `products.html`). |
| **`static/`** | **Статические файлы.** Содержит ресурсы, которые напрямую отдаются браузеру: CSS, JavaScript и изображения. |
| **`static/css/`** | Файлы стилей CSS (например, `style.css`). |
| **`static/js/`** | Файлы JavaScript (например, `main.js`). |
| **`static/img/`** | Изображения и иконки (например, `logo.png`, `favicon.ico`). |
| **`.xlsx`**| Предоставленные файлы
